🎯 Sélection de mission

🗻 Mission : Survol des cols frontières

Type : Surveillance

Description : Les drones de la PAF sillonnent les cols alpins à la recherche de passages clandestins, notamment la nuit.

Enjeux : Surveillance des passages clandestins par drones, surtout de nuit. Efficace pour repérer mais limité dans l’intervention. Forte symbolique sécuritaire dans les territoires frontaliers sensibles.

“Le drone, c’est comme une patrouille silencieuse. Mais il ne remplacera jamais l’œil du flic dans le col.” – Responsable PAF

🌊 Mission : Repérage embarcations

Type : Sauvetage & contrôle

Description : Les drones maritimes détectent les embarcations en détresse, déclenchant parfois des sauvetages, parfois des interceptions.

Enjeux : Repérage d’embarcations via drones maritimes. La détection ne garantit pas toujours un secours. Cela soulève des critiques sur la non-assistance en mer et le non-respect du droit maritime international.

“On n’arrive plus à séparer l’aide humanitaire de la surveillance.” – ONG Méditerranée

🏙️ Mission : Tests IA reconnaissance faciale

Type : Technologique

Description : Des essais menés dans des villes européennes sur l’usage de l’IA pour reconnaître les individus dans l’espace public.

Enjeux : Tests de reconnaissance faciale. Sujet à controverse pour ses implications en matière de vie privée et de discrimination algorithmique. Des ONG dénoncent ces expérimentations comme intrusives.

“Ces tests interrogent les libertés publiques.” – Rapport CNIL 2024

🛰️ Mission : Suivi des mouvements en Europe Centrale

Type : Dispositif Frontex

Description : Cartographie des flux migratoires avec caméras thermiques, outils prédictifs et veille permanente.

Enjeux : Surveillance prédictive via drones thermiques et IA en Europe centrale. Frontex centralise les données migratoires mais est accusée de pratiques opaques et de refoulements illégaux.

“Nous analysons les données en temps réel pour adapter nos réponses.” – Communiqué Frontex
